## Dendencies
- Python 2.7
- Tensorflow 1.0.0
- Numpy
- OpenCV 3.1.0*
- Pillow 3.4.2
- scipy 0.18.1
- umpy 1.11.2

## Install
- models download
    - [Models](https://drive.google.com/drive/folders/0B9jhaT37ydSyRk9UX0wwX3BpMzQ) download to ```styletransfer1/models/```

- server setup
```
    pip install -r requirements.txt
```

- test client settup
```
    test-views> npm install
```

## Run
```
   python server.py
```
